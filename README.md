# How-Humans-Explain-the-Difference-in-the-Quality-of-Plans----A-User-Study-Data-
The data for the paper How Humans Explain the Difference in the Quality of Plans -- A User Study - Krarup et al., ICAPS 2026.

The data includes the study that participants were tassked with completing. The cleaned results of the study coded. The planning problems for each task in the study as well as the constrained problems that were the results of the questions users were imagined to ask.
